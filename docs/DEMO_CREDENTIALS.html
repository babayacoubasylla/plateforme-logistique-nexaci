Comment se connecter (Web)
1) Ouvrir https://nexaci-frontend.onrender.com
2) Saisir Email OU Téléphone + Mot de passe
3) Valider → redirection vers le dashboard selon le rôle
Comment se connecter (Mobile Android)
1) Télécharger l’APK (Client ou Livreur) via le QR ou le lien
2) Autoriser « Sources inconnues » si demandé
3) Ouvrir l’app → Saisir Email/Téléphone + Mot de passe
4) Client: peut aussi « Créer un compte » depuis l’app
Test API (exemple)
Endpoint: POST https://nexaci-backend.onrender.com/api/auth/login
Body (JSON): { "email": "client1@example.com", "password": "client123" }
Variante téléphone: { "email": "0101010101", "password": "client123" }
Les numéros de téléphone sont normalisés automatiquement (070..., +225070..., 225070...). Toutes les variantes raisonnables sont acceptées à la connexion.
